Dynamic Reorderer

Below is a set of reorderable elements with the ability to dynamically create additional elements that are added to the set. This is a demostration to verify the fix for FLUID-1702.

Instructions

  1. Move one of the reorderable items
  2. Activate the "New Item" button, to create a new reorderable item
  3. Move the new item to a new position in the list
