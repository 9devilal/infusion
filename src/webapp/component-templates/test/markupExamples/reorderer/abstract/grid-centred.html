Div Grid Horizontally Centred

This page is an example of a grid with variably-sized cells, marked up for use with the Reorderer.

The point of this example is to show a case where one row in the grid has fewer columns than others. In this example, the third row has but four items centred horizontally, whereas the other rows have five. In this situation, an item in the shorter row is centred below two items in the row immediately above; and is itself above a pair of items in the row below it. Cells that have been tagged as reorderable are highlighted.
