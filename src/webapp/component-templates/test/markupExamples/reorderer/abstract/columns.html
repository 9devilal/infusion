Multiple Columns

This page is an example of columns, marked up for use with the Reorderer. The columns are implemented as a single-row table. In this example, divs inside table cells are marked as orderable. Contents that have been tagged as reorderable are highlighted.

In this mark-up, the Reorderer has been applied to the items using the GridLayoutHandler. Ultimately, a ColumnLayoutHandler will be created.

Due to bugs in dojo's Drag and Drop implementation, this example does not function properly. To experiment with the keyboard functionality, you should disable the Reorderer's use of Drag and Drop by editing Reorderer.js and commenting out the call to _enableDragAndDrop().
