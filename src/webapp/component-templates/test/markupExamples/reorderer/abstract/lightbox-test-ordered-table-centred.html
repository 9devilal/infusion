A Table with different number of items per row

This page is an example of a table in which one row in the grid has fewer columns than others, marked up for use with the Reorderer. The third row below has but three items centred horizontally, whereas the other rows have six. In this situation, an item in the shorter row is centred below two items in the row immediately above; and is itself above a pair of items in the row below it. Contents that have been tagged as reorderable are highlighted.
