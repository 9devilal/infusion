Unordered List (vertical)

This is a simple example of using the Reorderer with a list. If you look at the markup, you'll notice the "orderable" class. This is the contract between the Reorderer and its template: any element with an "orderable" class can be moved with the use of the Reorderer. If you look at the page source, you'll also notice a script block containing: new fluid.Reorderer({orderChangedCallback: function() {}, layoutHandler: new fluid.ListLayoutHandler() }, "list1"); This is the script that creates a Reorderer using a ListLayoutHandler and attaches it to this list.
