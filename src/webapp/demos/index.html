Infusion Components

These components are available in the Infusion framework download

Production

Layout Reorderer

View Demo | More Info

The Layout Reorderer provides drag and drop customization of modules in a portal environment. This component is part of the Reorderer family of components.

List Reorderer

View Demo | More Info

The List Reorderer provides reordering of elements in a list through drag and drop and/or numbering. This component is part of the Reorderer family of components.

Grid Reorderer

View Demo | More Info

The Grid Reorderer is a convenience function for applying the Reorder to a grid of items with minimal effort.

Image Reorderer

View Demo | More Info

The Image Reorderer is a JavaScript-based user interface component that lets users reorder images within a collection. This component is part of the Reorderer family of components.

Simple Text Inline Edit

View Demo | More Info

Simple Text Inline Edit allows a user to do quick edits to simple text without having to switch modes or screens. All work is done on the same interface which helps the user maintain context. This component is part of the Inline Edit family of components.
