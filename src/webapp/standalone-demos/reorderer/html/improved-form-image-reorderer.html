Image Reorderer Example

Image Reorderer: To use the keyboard: Screen reader users turn off virtual cursor (insert + z on JAWS). Select an image using i, k, m or j. Move an image using CTRL + i, k, m or j. View an image by pressing Enter.
Move an image by clicking on it and dragging it to a new location. To use the keyboard:
  • Select an image using i, k, m or j, or the arrow keys.
  • Move an image using CTRL + a direction key.
  • View an image by pressing Enter.
